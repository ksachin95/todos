# Todo App

A simple, responsive todo application built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as complete
- Delete tasks
- Persist tasks in local storage
- Toggle between light and dark themes

## How to Run

1. Open `index.html` in your browser, or
2. Serve the project with a local static server (for example, VS Code Live Server)

## Project Structure

- `index.html` — app layout
- `styles.css` — styling and theme behavior
- `script.js` — task logic and local storage handling
